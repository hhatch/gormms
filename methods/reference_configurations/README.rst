***************************************
Reference Configurations (Ref. Config.)
***************************************

Reference configurations report the potential energy and other properties of a single configuration as described in the `NIST Standard Reference Simulation Website <https://www.nist.gov/programs-projects/nist-standard-reference-simulation-website>`_ of :footcite:t:`shen_nist_2024`.
These results are expected to be accurate to machine precision and do not involve statistical averages.
